# ⚔️ Collection Quest

Python の**コレクション型**（`list` / `dict` / `set` / `tuple` / `str`）のメソッドを、
**進化する勇者**と一緒に **RPG風クイズ**で学べる Web アプリです。

## 🎮 今すぐ遊ぶ（インストール不要）

### 👉 https://kamurai42tokyo.github.io/collection-methods-quiz/

スマホ・PC どちらでも、**上のリンクをブラウザで開くだけ**で遊べます。
アカウント登録・インストール・サーバー準備は一切不要です。誰でもそのまま利用できます。

> 📱 **スマホアプリのように使う**: ブラウザのメニューから「ホーム画面に追加」すると、
> アイコンから起動でき、オフラインでも動きます（PWA対応）。

## 🕹 遊び方

1. **たたかう型**（list / dict / set / tuple / str）を選ぶ
2. **出題形式**（動作当て / 名前当て / 出力予想）と**問題数**（5 / 10 / 20問）を選ぶ
3. 「ぼうけんに でる」でスタート。正解するたびに勇者がレベルアップして進化！

## ✨ 特長

- **🦸 進化する歩く勇者**: 正解数＝レベル。1問正解するごとに勇者が1段階進化（最大Lv20）。
  村人 → 戦士 → 騎士 → 勇者 → 伝説の勇者へ。武器・鎧・兜・マント・オーラが増え、常に歩き続けます（SVGベクター描画・外部画像なし）。
- **🎮 RPG風UI**: メッセージウィンドウ、コマンドメニュー風の選択肢、EXPバー、撃破数、称号、レベルアップ演出。
- **3つの出題形式**
  - 動作当て: メソッド → 動作を選ぶ
  - 名前当て: 動作 → メソッドを選ぶ
  - 出力予想: コード → 出力結果を選ぶ
- **選択肢は毎回ちがう**: 正解＋ダミーを出題のたびにプールから抽選・シャッフル。位置や顔ぶれで覚えられません。
- **出題範囲を選べる**: 型・形式・問題数を自由に組み合わせ。問題数だけ最大レベルが伸びます。
- **直近の問題を避ける**: `localStorage` に履歴を保存し、出題の偏りを軽減。
- **スマホ最適化**: レスポンシブ + PWA。`prefers-reduced-motion` でアニメ停止にも対応。

## 🛠 開発者向け

バックエンド不要・静的ファイルのみ（HTML / CSS / バニラ JS）で動きます。

### ローカルで動かす

データを JS に埋め込んでいるため、`index.html` をブラウザで開くだけで動きます。

```bash
# 方法1: ファイルを直接開く
open index.html        # macOS

# 方法2: 簡易サーバー（PWAも確認できる）
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

### 公開（GitHub Pages）

このリポジトリは GitHub Pages で公開しています。`main` に push すると自動で再ビルドされ、
上記URLに反映されます。新規に公開する場合は **Settings → Pages → Source: `main` / `/ (root)`** を選択します。

### 構成

```
App/
├── index.html        画面の骨組み
├── css/style.css     スタイル（RPGテーマ・歩行アニメ・モバイル優先）
├── js/
│   ├── data.js       メソッド定義（ここに追記すれば出題が増える）
│   ├── hero.js       進化する勇者のSVG描画（レベル別の装備・歩行・レベルアップ）
│   └── app.js        クイズの出題・採点ロジック
├── manifest.json     PWA 設定
├── icon.svg          アプリアイコン
└── README.md
```

### メソッドを追加するには

`js/data.js` の配列に1要素追加するだけです。

```js
{ type: "list", name: "append", signature: "list.append(x)", category: "追加",
  mutating: true, returns: "None",
  description: "末尾に要素を1つ追加する",
  example: "lst = [1, 2]\nlst.append(3)\nprint(lst)", result: "[1, 2, 3]" }
```

### 勇者の見た目を変えるには

`js/hero.js` の `TITLES`（称号）と `LEVELS`（各レベルの装備色・武器・マント・兜・オーラの定義テーブル）を編集します。
